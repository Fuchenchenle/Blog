1.注释

注释单行使用 // 注释多行以 /* 开头 */ 结尾

2.常量

在程序运行时，不会被修改的量

常量类型 说明 举例
字符串常量 用双引号括起来的内容 “Hello world”,”你好 世界”
整数常量 不带小数的数字 666，-888
小数常量 带小数的数字 13.14，5.21
字符常量 用单引号括起来的内容 ‘A’,’0’,’’我
布尔常量 布尔值，表示真假 true,flase
空常量 一个特殊的值，空值 null

3.数据类型

Java基本类型共有八种，基本类型可以分为三类，

1.字符类型char（占用2字节）

2.布尔类型boolean(1)以及数值类型byte(1)、short(2)、int(4)、long(8)、float(4)、double(8)

3.数值类型又可以分为整数类型byte、short、int、long和浮点数类型float、double

精准度由低到高排序：

byte,short,char > int > long > float > double

4.变量

在程序运行过程中,其值可以发生改变的量

格式： 数据类型 变量名 = 变量值

int a = 10;
loog 1 = 10000000L;          //定义long类型防止变量过大，需要在变量值后面加L
float f = 3.1415f            //定义Float类型需要在变量值后面加F

运算符优先级：

优先级 运算符 结合性
1 ()、[]、{} 从左向右
2 !、+、-、~、++、– 从右向左
3 *、/、% 从左向右
4 +、- 从左向右
5 «、»、>>> 从左向右

9 ^ 从左向右
10 | 从左向右
11 && 从左向右
12 || 从左向右
13 ?: 从右向左
14 =、+=、-=、*=、/=、&=、|=、^=、~=、«=、»=、>>>= 从右向左
7.1 算术运算符

对常量或者变量进行操作的符号

表达式：用运算符把常量或者变量连接起来符和java语法的式子，不同运算符连接的表达式体现的是不同类型的表达式。

int a = 10;
int b = 20;
int c = a + b;       //+ 是运算符，并且是算术运算符   a+b 是表达式，由于＋是运算符，所以这个表达式叫算术表达式

整数相除只能得到整数，要想得到小数，必须有浮点数的参与

符号 作用 说明
＋ 加 得到的是和
- 减 得到的是差
* 乘 得到的是积
/ 除 得到的是商
% 取余 得到的是余数 
int 1 = 10;
char c = 'A'；        //'A'的值是65
c = 'a';            //'a'的值是97
system.out.println(1+c)     //字符在＋操作的时候是用计算机底层对应的数值来计算的，具体参照ASCLL码

字符串“+”操作

System.out.println("黑马"+ 6 + 66 )    //当+操作出现在字符串，找个+是字符串连接符    
System.out.println(1 + 99 + "年黑马")   //连续出+操作，从左到右逐个执行

7.2 赋值运算符

赋值运算符是指为变量或常量指定数值的符号 “=”，赋值的意思是将等号右边的值赋予等号右边的变量

符号 作用 说明
= 赋值 a=10,将10赋值给变量a
+= 加后赋值 a+=b,将a+b的值给a
-= 减后赋值 a-=b，将a-b的值给a
*= 乘后赋值 a*=b，将a×b的值给a
/= 除后赋值 a/=b，将a÷b的商给a
÷= 取余后赋值 a%=b，将a÷b的余数给a

​

7.3 自增自减运算符
符号 作用 说明
++ 自增 变量的值加1
– 自减 变量的值减1 
public class demo{
    public static void main(Sting[] args) {
        int i = 10;
        int j = i++;     //++在后面，先把变量值10赋值给J，后做自增运算，控制台j输出的是10
        int j = ++i;    //++在前面，先做自增运算，后把运算的结果赋值给j，控制台输出的是11
        System.out.println(j)
    }
}
7.4 关系运算符

关系运算符的结果都是boolean类型，要么是true，要么是false

符号 说明
== a==b,判断a和b的值是否相等，成立为true，不成立为false
!= a!=b,判断a和b的值是否不相等，成立为true，不成立为false
> a>b,判断a是否大于b，成立为true，不成立为false
>= a>=b,判断a和b是否大于等于b，成立为true，不成立为false
< a<=b,判断a是否小于b，成立为true，不成立为false
<= a<=b,判断a是否小于等于b，成立为true，不成立为false
7.5 逻辑运算符

逻辑运算符是用来链接关系表达式的运算符。

逻辑运算符的优先级：

！大于 && 大于 ||

符号 作用 说明
&& 逻辑与 当两个操作数都为true时，结果为true
|| 逻辑或 两个操作数有一个为true时，结果为true
! 逻辑非 对操作数的值取反
7.6 三元运算符

格式：关系表达式？表达式1：表达式2；

范例：a>b?a:b

计算规则：首先计算关系表达式的值

​ 如果值为true，表达式1的值就是运算结果

​ 如果值为false，表达式2的值就是运算结果

public class Demo{
    public static void main(String[] args){
        int a = 10;
        int b = 20;

        int max = a > b ? a : b        //获取两个数据中的较大值
    }
}

10.4 三种循环的区别

for循环和while循环先判断条件是否成立，然后决定是否执行循环体（先判断后执行）

do…while循环先执行一次循环体，然后判断条件是否成立，是否继续执行循环体（先执行后判断）

13.数组

13.1 数组定义格式

数组是一种用于存储多个相同类型的数据存储模型

//格式一：数据类型[] 变量名
         int[] arr    //定义了一个int类型的数组，数组名是arr
//格式二：数据类类型 变量名[]
          int arr[]    //定义了一个Int类型变量，变量名是arr数组
13.2 数组初始化

java中的数组必须初始化后才能使用，初始化就是为数组元素分配内存空间，为每个数组元素赋值

动态初始化：初始化时只指定数组长度，由系统为数组分配初始值

// 格式：数据类型[] 变量名 = new 数据类型[数组长度];
        int[] arr = new int[3];

静态初始化：初始化时指定每个数组元素的初始值，由系统决定数组长度

//格式：数据类型[] 变量名 = new 数据类型[]{数据1，数据2，数据3，...};
        int[] arr = new int[]{1,2,3};
//简化格式；数据类型[] 变量名={数据1，数据2，数据3，...}；
        int[] arr={1,2,3};

格式：数组名[索引] 索引从0开始，是连续的，索引每次逐一增加1

13.4 常见操作

​

1.遍历

遍历获取数组中元素

使用for循环来遍历 自增 i++

int[] arr={11,22,33};
for(int x =0; x<3; x++){    //x<3 数字3是数组的个数
    System.out.println(arr[x]);
}

如果元素过长，可以使用.length来获取元素数量

格式：数组名.length

int[] arr={11,22,33};
for(int x =0; x<arr.length; x++){    //x<3 数字3是数组的个数
    System.out.println(arr[x]);
}

2.获取最值

int[] arr = {12,45,98,73,60};
//思路：
int max;    //定义一个变量，保存最大值    
int max = arr[0];    //获取数组中第一个数据作为变量的初始值
for(int x =1;x<arr.length;x++){   //与数组中剩余的数据逐个对比
    if(arr[x] > max){    //如果数组arr中的数数值比max大
        max = arr[];    //将arr中的数值赋值给max
    }
}
